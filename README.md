# 🚀 Task Manager - Full Stack Project

Este es un gestor de tareas moderno construido con un backend en **FastAPI** y un frontend en **React** con **Tailwind CSS v4**.

## 🛠️ Tecnologías Utilizadas

### Backend
* **Python 3.10+**
* **FastAPI**: Framework web de alto rendimiento.
* **Uvicorn**: Servidor ASGI para producción y desarrollo.

### Frontend
* **React 18**: Biblioteca para interfaces de usuario.
* **Vite**: Herramienta de construcción ultra rápida.
* **Tailwind CSS v4**: Framework de estilos basado en utilidades.
* **Axios**: Cliente HTTP para conectar con la API.
